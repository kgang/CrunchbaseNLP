1.0 Introduction¶

Crunchbase is an aggregator of information related to startups, including descriptions of their business, founder bios, and funding events. We'll be using the Crunchbase 2013 Snapshot to analyze the long descriptions of the companies using NLP and topic modeling. We'll see what classifications exists within software startups and if that information can be used to new companies.

The dataset is available through a free Crunchbase Data account here and represents historical Crunchbase data collected before December 2013. It is licensed under Creative Commons with Attribution CC-BY. © 2013

All content and files can be found on my repo here.

import re

def clean_text(text):
    """Removes emails, urls, text in parentheses, and \r or \n.
       Replace dashes in words with spaces(e.g. easy-to-use > easy to use)"""
    text = re.sub('\S*@\S*\s?', '', text)
    text = re.sub('\S*\.\S+','',text)
    text = re.sub('\(.*?\)','', text)
    text = re.sub('\\n|\\r','',text)
    text = re.sub('(?<=\S)-',' ', text)
    return text

def space_or_punct(token):
    """Helper function to identify space and punctuation."""
    return token.is_punct or token.is_space

def pronoun_or_person(token):
    """Helper function to identify pronouns and people."""
    return (token.lemma_ == '-PRON-') or (token.ent_type_ == 'PERSON')

def lemmatizer(overview):
    """Lemmatizes and removes punctuation and people."""
    overview = [token.lemma_ for token in overview 
                if not space_or_punct(token) and not pronoun_or_person(token)]
    doc = u' '.join(overview)
    return nlp.make_doc(doc)

def remove_stopwords(overview):
    """Returns text for modeling."""
    overview = [token.text.lower() for token in overview if not token.is_stop]
    return overview

nlp.add_pipe(lemmatizer, name='lemmatizer', after='ner')
nlp.add_pipe(remove_stopwords, name='rm_stopwords', last=True)
